Подробное описание

Шифрование методом табличной маршрутной перестановки

Ключ устанавливается в конструкторе. Для зашифровывания и расшифровывания предназначены методы encrypt и decrypt.

Предупреждения
Реализация только для русского языка

Класс, реализующий табличный шифр. Класс реализует шифр табличной маршрутной перестановки.

Метод шифрования текста с помощью табличной маршрутной перестановки.

Записывает текст в таблицу на основе полученного ключа и вычисленного значения количсества строк таблицы. Маршрут записи: по горизонтали слева направо, сверху вниз. Маршрут считывания: сверху вниз, справа налево.

Валидация ключа.

Функция проверяет, что строка ключа не пуста и содержит только цифры. Если строка валидна, она преобразуется в целое число. В случае ошибки выбрасывается исключение.
